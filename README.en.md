# 缓存关系管理工具包

#### Description
用于解决使用了缓存的JAVA项目中普遍存在的数据对象互相影响，数据对象的变化难以及时的、容易的影响到被缓存的数据对象问题。例如：方法1引用了A、B、C三张表的数据，方法2引用了B、D两张表的数据，方法3引用了A、C两张表的数据，当其中某个表的数据发生变化时，传统的方法难以控制各方法的缓存失效。本项目致力于解决以上问题
